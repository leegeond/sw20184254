# sw20184254 Markdown 문법
1. "#" 개수에 따라 제목 부제목 순으로 줄어든다
# like
## like
### like
2. 글을 분리할때는'-'를 사용한다
-------------
이
------------
렇
------------
게
----------
3.인용문 작성은 '>'를 사용한다 개수에 따라 칸도 늘어난다
>안녕하세요
>>안녕하세요
>>>안녕하세요
4. 링크 첨부 가능하다 
[Google](https://www.google.com)
5. *를 이용하면 강조도 할수있다
ex)**20184254**
